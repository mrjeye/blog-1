升级android studio 3.1之后出现apk/app-debug.apk does not exist on disk.

发表于 2018-03-31 | 分类于 android |

其他版本也会遇到这个问题，在你尝试过重新ide，clean项目等都无效后可以确认下是不是这个问题。
选择ide的Run -> Edit Configurations…。在Before launch列表里看下是不是只有一个Instant App Provision。如果是新建一个Gradle-aware Make. Task填写空就行。

查看是不是只有一个Instant App Provision

新建一个Gradle-aware Make

点击加号

不用输入名字直接点击OK

点击OK完成设置就可以正常Run了
